What is HTML?
HTML stands for Hyper Text Markup Language
A markup language is a set of markup tags
HTML documents are described by HTML tags
Each HTML tag describes different document content

HTML Tags HTML tags are keywords (tag names) surrounded by angle brackets: <tagname>content<tagname>
HTML tags normally come in pairs like <p> and </p> The first tag in a pair is the start tag/ Opening Tag, the second tag is the end tag/ Closing Tag
The Closing tag is written like the Opening tag, but with a slash before the tag name
